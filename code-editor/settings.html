Settings

Configure the editor appearance, LLM integration, and local development tools. All settings are stored as a JSON object that you can import, export, and share across machines.

Appearance

Show Line Numbers
Display line numbers in the editor gutter.
Show Minimap
Display a code minimap in the right margin for quick navigation.
Enable Code Folding
Allow collapsible code blocks via fold markers in the gutter.
